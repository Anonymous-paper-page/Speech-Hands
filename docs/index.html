Abstract

We present a voice-agentic framework that learns one critical omni-understanding skill: knowing when to trust itself versus when to consult external audio perception. Naive joint fine-tuning on automatic speech recognition and audio reasoning often degrades performance, because the model is misled by noisy hypotheses from external perception models. We re-cast the problem as an explicit self-reflection decision: the model learns to emit a discrete routing token that decides, per example, whether to answer directly from audio, defer to an external candidate, or re-derive the answer from both. The resulting framework yields 12.1% relative WER improvement across seven OpenASR benchmarks and 77.37% accuracy on DCASE 2025 AudioQA.

Self-Reflection Primitive

For every input, Speech-Hands emits one of three learnable routing tokens before producing its final answer. The token makes the perception-vs-reasoning trade-off explicit and trainable:

<internal>

Trust self

The internal prediction is reliable; ignore the external candidate and answer directly from audio.

<external>

Defer

The external ASR / audio-reasoning model's top candidate is better; adopt it.

<rewrite>

Re-derive

Neither the internal nor the external prediction is correct; re-derive the answer using audio and the external candidate list as grounding context.
